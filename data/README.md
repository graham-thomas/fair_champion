# This directory contains data files used in the fair_champion project
It contains:
- Examples of the newletter
- Parsed publication lists from the newsletters
- A single example publication input file 'test-1.docx' used for initial testing of the tool
- A medium complexity input file with 5 publications 'test-5.docx'
- An example output file to show what I want the tool to produce 'manually_curated_example_output.xlsd'
- A copy of the prompt I used to generate the initial version of the tool 'prompt.docx'
